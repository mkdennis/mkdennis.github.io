Week 4: GE Smart Refrigerator Mobile App

For now, you can assume the application will run on the current generation iPad (9.7″ Retina display with 2048×1536 resolution)."

Please proceed to develop initial user interface concepts for the GE-IQ Smart Fridge Mobile App. As background, the GE-IQ Smart Fridge series refrigerators will be an internet-connected fridge with a 22” diagonal touchscreenThe initial release of the mobile app will enable users to perform the following functions at a minimum:

- Monitor and adjust the refrigerator and freezer temperature

- View a live video feed of the inside of the fridge

- View, modify, and create shopping lists

- Write a custom note that will appear on the fridge’s display

- View a list of expiry reminders

- Control Spotify playback
